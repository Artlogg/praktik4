# Практические работы 7-8: React

Проект создан в формате Vite + React.

## Запуск

```bash
npm install
npm run dev
```

После запуска открыть адрес, который покажет Vite, обычно:

```text
http://localhost:5173
```

## Практическая работа 7

Реализовано:

- отдельный компонент `ProfileCard`;
- подключение компонента в `App.jsx`;
- JSX-разметка;
- данные студента: имя, специальность, группа, описание;
- загрузка изображения для аватарки через `input type="file"`;
- использование HTML-элементов `section`, `div`, `h1`, `h2`, `p`, `ul`, `li`, `label`, `input`, `img`.

## Практическая работа 8

Реализовано:

- отдельный компонент `ApiUsers` для запроса к API;
- отдельный компонент `UserCard` для отображения одного элемента списка;
- запрос к тестовому API `https://jsonplaceholder.typicode.com/users` при открытии страницы;
- сохранение полученных данных в состояние `users` через `useState`;
- состояние загрузки `loading` с текстом `Загрузка...`;
- состояние ошибки `error` с текстом `Ошибка загрузки данных`;
- вывод списка пользователей через `map()`;
- уникальный ключ `key={user.id}` для каждого элемента списка.
